### Setup ###
* install NodeJS and MongoDB.
* install dependencies: run ```npm install```

### Start server ###
* Start mongoDB server: run ```mongod```
* Start server: run ```node app.js```
* Start server with Nodemon (for development): run ```nodemon app.js```
